User Interface Programmer

My Job

My job in the project was mainly to code all necessary tools for the user interface to work, at the same time i worked on scrpting and mounting many scenes during developement.

Image Filler

Image filler, horitzontal and vertical, was mainly done for uses like hp bar and mana bar for the hud, and later on we used it aswell when creating slider.

Used aswell in the hud as feedback to let the player know how damage and stamina is taking.

Scripting

At the beggining of the the project i worked on cooldown script and hp and stamina ones.

During developement i worked on creating most ui related scenes scripts, and menus, where we can change display settings, show fps, activate controller vibration or VSync among other things.

We have an Audio menu aswell where we can use the volume slider to change the volume of the game or a checkbox to mute it.

Slider & CheckBox

Coded Slider and CheckBox for later creating display menu and audio menu, and let the player change window state, deactivate controller vibration or game volume.

Menus & Scenes

Worked on scene scripting through milestones.

Once we start the game introducing any input, Lore scene introduces de player in a scrolling text to start playing the tutorial.
