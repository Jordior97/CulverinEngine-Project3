Project Overview

At first I did a little try in some 2D texturing for the stage scrum, but after a week, a extra artist was needed in the UI scrum so I was moved to the UI scrum to work in the HUD art. Finally I ended being the only artist in UI and I was in charge of almost all the UI art, including menus, charge screens, HUD, etc...

Tileable Floor Texture

I did a tileable floor texture, handpainting with 3D-Coat.

UI HUD

When I arrived UI scrum I was in charge of the ability chart and the character icons. Later, when I became the only UI artist, I was also in charge of the Life and Stamina/Mana bars and all the new HUD elements that we added later. I created all this elements drawing and using photoshop.
